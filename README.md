# Vite Starter Template

A comprehensive starter template built with Vite and the latest web technologies.

## 🚀 Features

- **React 18** - Latest React with concurrent features
- **TypeScript** - Full type safety throughout the app
- **Vite** - Lightning fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Declarative routing for React
- **React Query** - Powerful data fetching and caching
- **Context API** - Built-in state management
- **Supabase** - Backend as a service with authentication
- **Vapi** - Voice AI integration
- **MUI Charts** - Beautiful and interactive charts
- **React Flow** - Node-based editor and workflow builder
- **Framer Motion** - Smooth animations and transitions

## 📦 Installation

1. Clone the repository:
\`\`\`bash
git clone <repository-url>
cd vite-starter-template
\`\`\`

2. Install dependencies:
\`\`\`bash
npm install
\`\`\`

3. Copy environment variables:
\`\`\`bash
cp .env.example .env.local
\`\`\`

4. Update environment variables with your API keys:
- Get Supabase credentials from [supabase.com](https://supabase.com)
- Get Vapi API key from [vapi.ai](https://vapi.ai)

5. Start the development server:
\`\`\`bash
npm run dev
\`\`\`

\`\`\`

The `dist` folder contains the production build.
